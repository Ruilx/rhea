# The Project Rhea

![rhea logo](data/assets/rhea-inv-alpha.png)

Rhea is a HTTP server interface that based on FastAPI and provides a simple, 
flexible, and fast framework written in Python.

Rhea，FastAPI的通用HTTP API接口项目

## Features

1. Python 3.10+
2. 基于FastAPI作为ASGI框架
3. 应用事件循环
4. 优美的数据库操作调用
5. 动态路由
6. 中间件插件和钩子回调
7. 异常处理
8. 优雅进退
